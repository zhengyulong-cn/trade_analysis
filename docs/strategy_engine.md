# 交易系统指标

## 线段生成

依靠charting_library图表库，其自带指标引擎，也就是说会从左往右对每个K线执行。

指标作用：当价格上穿EMA20且和前面下跌段间隔5根K线以上，构建上涨段；当价格下穿EMA20且和前面上涨段间隔5根K线以上，构建下跌段。

当K线没有EMA20时候，不执行，只有K线有EMA20时候才开始执行。起点K线收盘价若低于EMA20，则认为前面是下跌段，下跌段的终点则是该K线最低价；起点K线收盘价若高于EMA20，则认为前面是上涨段，上涨段的终点则是该K线最高价。当然实际情况，这个认为的线段是不必画出的，只是作为计算的参照。

假设前面线段是下跌段，从下跌段终点K线往右执行遍历。

- 若新K线最低价低于EMA20且价格低于下跌段终点，则下跌段终点移动到该K线最低价。
- 若新K线最低价低于EMA20但价格高于下跌段终点，不操作。
- 若新K线最高价高于EMA20但和下跌段终点间隔5根K线以内，不操作，下跌段维持。
- 若新K线最高价高于EMA20但和下跌段终点间隔5根及5根以上，下跌段结束，开始画上涨段，上涨段的起点就是前面下跌段的终点K线最低价，上涨段终点则为当前K线最高价。

在遍历过程中，会维护一个线段数组，这个是根据K线更新的。
