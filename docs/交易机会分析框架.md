# 交易机会分析框架

## Step1：确定4H线段方向

视角1：30F的EMA20上穿下穿EMA120构成4H线段方向的判断

视角2：观察30F和EMA20的阻力关系，对于两个连续的相反30F线段，如果线段a对EMA20表示没有阻力，而线段b对EMA20表现有阻力，那么4H线段方向就是线段a的方向。【主观交易，代码不实现】

## Step2：判定30F结构在4H方向是否发生动能衰竭

30F结构没有发生动能衰竭，并不是说当前30F线段就不会终结，而是大概率会在回调后走出新的同向30F线段。

30F结构发生动能衰竭，说明会出现两种情况：

- 4H线段方向可能反转
- 4H线段方向延续，但当前位置附近会形成交易区间

> 注意：当前30F是否终结和30F结构动能衰竭没有关键，30F是否终结是看5F结构是否动能衰竭。

## Step3：确定操作视角

只要30F结构没有动能衰竭，4H方向就是操作视角方向，无论当前的30F线段是趋势推动段还是趋势回调段，4H方向就是操作视角方向。

如果30F结构发生动能衰竭，操作视角方向则是和4H线段方向相反。

## Step4：确定当前30F线段类型并制定入场方式

30F结构没有动能衰竭，那么肯定是下面两种情况，且类型固定：

趋势推动段

趋势回调段

30F结构没有动能衰竭，那么会是下面情况：

区间内部段

区间内部段转趋势推动段
